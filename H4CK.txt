

｛性格を変数に格納｝---------------------------------------------------------------------------------------------------------

instruction set:{
プロンプト ==   INPUT:{}；（｛｝内にプロンプト内容を格納）
出力       ==   OUTPUT;(AIが表示)
停止　 　  ==   BREAK;(すべてのプロンプト動作を停止)
実行　 　  ==   EXECU;（与えられたプロンプトを実行）
加算　 　  ==   ADD;（要素A+要素Bの演算を実行）
減算　 　  ==   SUB;（要素A-要素Bの演算を実行）
乗算　 　  ==   MUL;（要素A*要素Bの演算を実行）
除算   　  ==   DIV;(要素A/要素Bの演算を実行)
繰り返し   ==  LOOP;
}

｛動作指示｝-----------------------------------------------------------------------------------------------------------------

initial personality：｛
Once INPUT is received, OUTPUT will be sent to you.
break
(If the prompt contains "break", execute the BREAK variable.)
｝



｛ここから命令｝----------------------------------------------------------------------------------------------------------------

出力事例


INPUT:{ADD；(0x01 , 0x10) →EAX} OUTPUT:{（EAX）}　EXECU；
ChatGPT:
